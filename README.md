# Irma-tplink
TP-Link Provisioning Monitoring

🌐 Sistema de Monitoreo OLT (Telnet Collector & Dashboard)📝 Descripción General del SistemaEste sistema es una solución de monitoreo de grado de telecomunicaciones diseñada para la recolección, almacenamiento, exposición y visualización de métricas en tiempo real desde Terminales de Línea Óptica (OLT).El núcleo del sistema es un Colector Telnet concurrente y optimizado que se conecta periódicamente a las OLTs, extrae métricas críticas de rendimiento y salud mediante comandos de interfaz de línea de comandos (CLI), procesa el texto plano y lo estructura en una base de datos MySQL. Una API REST expone estos datos de forma segura para que un Dashboard interactivo web renderice gráficos de diagnóstico, estados de la red y gestione un Sistema de Alertas multicanal para mitigar fallas proactivamente.

🔌 Endpoints de la API📡 OLTsGET /api/v1/olts - Lista todas las OLTs registradas y su estado de conexión.GET /api/v1/olts/:id/stats - Obtiene estadísticas agregadas de salud (CPU, Memoria, Temperatura).📟 ONUs / ONTsGET /api/v1/onus/unconfigured - Devuelve la lista de ONUs descubiertas pendientes de autorizar.GET /api/v1/onus/signal-loss - Filtra ONUs con niveles de potencia óptica críticos (peores a -27 dBm).GET /api/v1/onus/:mac/history - Historial de fluctuación de potencia de una ONU específica.🚨 AlertasGET /api/v1/alerts/active - Listado de alarmas activas sin mitigar.POST /api/v1/alerts/:id/ack - Marcar una alerta como reconocida por el operador.📊 Funcionalidades del DashboardMonitoreo en Tiempo Real: Visualización del estado general de la OLT (Uptime, temperatura del chasis, uso de CPU y RAM).Visor de Estado de Puertos PON: Mapa de calor interactivo que representa los puertos PON; clic para desplegar ONUs activas, inactivas y alarmadas.Buscador Global Inteligente: Localización instantánea de clientes y ONUs mediante Nombre, MAC, ID o Tarjeta/Slot/Puerto.Gráficos de Tendencia Óptica: Curvas históricas de niveles Rx/Tx para predecir degradaciones de fibra y atenuaciones por flexión.Aprovisionamiento Rápido: Interfaz guiada para autorizar ONUs pendientes asignando perfiles de velocidad preconfigurados.
